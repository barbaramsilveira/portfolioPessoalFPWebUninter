💻 Projeto da Disciplina de Desenvolvimento Web
Este projeto foi desenvolvido como parte da disciplina do curso de Análise e Desenvolvimento de Sistemas da Uninter.

O objetivo foi aplicar conhecimentos práticos utilizando apenas as tecnologias HTML, CSS e JavaScript, sem o uso de frameworks externos.

📌 Requisitos do Projeto
A seguir, os requisitos definidos pela Universidade:

🔗 Link "Sobre Mim"
Uma página com informações pessoais, hobbies e interesses.

🎓 Link sobre Formação Educacional e Idiomas
Informações sobre sua trajetória acadêmica e idiomas que você domina.

🧰 Link para Portfólio
Página com links para projetos desenvolvidos durante o curso ou outros trabalhos relevantes. Exemplos utilizados durante a disciplina também podem ser incluídos.

✉️ Link para Contato
Página com formulário visual contendo os seguintes campos:
nome, email, mensagem.

🎨 Estilo com CSS puro
O uso de bibliotecas como Bootstrap ou similares não é permitido. Todo o layout deve ser construído com CSS puro.

💬 Comentários no Código
Explicações devem ser adicionadas no código por meio de comentários claros, porém objetivos — sem exageros.

🚀 Tecnologias Utilizadas
HTML5

CSS3

JavaScript (puro)
